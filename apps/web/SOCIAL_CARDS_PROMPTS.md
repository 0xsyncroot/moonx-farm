## üéØ AI Generation Best Practices

### **DO's:**
- ‚úÖ **Use negative prompts** to avoid text overload: `--no text-heavy --no cluttered --no busy-layout`
- ‚úÖ **Focus on 1-2 key visual elements** per card
- ‚úÖ **Emphasize spacing and minimalism**: "spacious layout", "clean design"
- ‚úÖ **Use visual metaphors**: arrows for swap, charts for portfolio, shields for security
- ‚úÖ **Brand consistency**: Always mention "MoonXFarm" + single feature name
- ‚úÖ **Use MoonXFarm Orange (#ff7842)** as primary accent color

### **DON'Ts:**
- ‚ùå **Don't request multiple text elements** in single prompt
- ‚ùå **Avoid complex layouts** with many elements
- ‚ùå **Don't use bullet points or lists** in visual prompts
- ‚ùå **Avoid requesting specific text placement** (let AI decide)
- ‚ùå **Don't combine too many features** in one card
- ‚ùå **Don't use blue as primary color** (use orange #ff7842)
